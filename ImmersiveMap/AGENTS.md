
# This is a geo rendering engine project.
ImmersiveMapFramework/ - folder contains map library.

# Description
The project uses mapbox api for vector tiles data.
Usages: Swift, Metal GPU.

Vector tile specification: documentation/TILE_SPECIFICATION.md

Documentation folder contains helpful information about implemented modules

# Main files
1. ImmersiveMapFramework/Renderer.swift - contains render frame logic.
2. ImmersiveMapFramework/Shaders - contains shaders
3. ImmersiveMapFramework/Camera - contains camera logic

# Code style
Follow swift, ios developers code style rules.
